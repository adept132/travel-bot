# Travel Bot 🗺️ - Ваш личный дневник путешественника

Telegram бот для записи и управления вашими путешествиями. Сохраняйте воспоминания о посещенных местах, добавляйте фотографии и создавайте интерактивные карты ваших приключений.

## 🌟 Основные возможности

- **📝 Управление путешествиями** - создание путешествий, добавление мест с автоматическим определением координат
- **🗺️ Визуализация** - heatmap карты с фильтрами по континентам, рейтингу и датам
- **📸 Мультимедиа** - добавление фотографий и видео (до 3 бесплатно, до 8 в премиум)
- **🏆 Геймификация** - система достижений с 17+ различными наградами
- **💎 Премиум функции** - расширенная статистика, умные фильтры, увеличенные лимиты
- **📊 Отчеты и экспорт** - детальные отчеты, полные архивы с фотографиями
- **🔍 Умный поиск** - поиск по местам, городам, фильтрация по странам и датам
- **🚀 Быстрое добавление** - быстрое добавление мест без создания полного путешествия
- **🔔 Умные напоминания** - настраиваемые уведомления о добавлении новых мест

## 🚀 Быстрый старт

### 1. Установка
```bash
git clone https://github.com/your-username/travel-bot.git
cd travel-bot
pip install -r requirements.txt
```
### 2. Настройка
Создайте файл travel_config.py:
```python
API_TOKEN = "your_telegram_bot_token_here"
admin_id = 123456789  # Ваш Telegram ID
mail = "your_email@example.com"
card = "your_card_number"
```
### 3. Запуск
```bash
python Traveler_bot.py
```
